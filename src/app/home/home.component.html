Welcome to the Dental Clinic Management System

To begin, input a valid api-key at the top right. Click "Test" to validate via the backend server.

If the backend server has been inactive for over an hour, it will be sleeping and will take some time to spin up.

A valid api-key must be present in the input box to successfully perform http requests.

Navigate to one of the four user interfaces using the navbar at the top of the screen.
